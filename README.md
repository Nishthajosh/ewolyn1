# Ewolyn Website

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Routes

| Path          | Page                     |
|---------------|--------------------------|
| `/`           | Home                     |
| `/services`   | Services                 |
| `/funding`    | Funding & Schemes        |
| `/about`      | About Us                 |
| `/impact`     | Impact & Statistics      |
| `/contact`    | Contact Us               |

## Build & Deploy

```bash
npm run build   # production build
npm start       # start production server
npm run lint    # ESLint
npm run typecheck  # TypeScript type-check
```

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS 3
- App Router (React Server Components)
